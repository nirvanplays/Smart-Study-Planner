# Smart Study Planner

Smart Study Planner is a cloud-based web application designed to help students manage study tasks, deadlines, and productivity efficiently.

## Features
- Google Authentication
- Add and manage study tasks
- Cloud Firestore database
- Productivity analytics and progress tracking
- Adaptive study planning
- Hosted using Firebase Hosting

## Technologies Used
- HTML, CSS, JavaScript
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting
- Firebase Analytics

## Live Demo
https://smart-study-planner-86f98.web.app

## How It Works
1. User logs in using Google Authentication
2. Adds study tasks with deadlines and hours
3. Data is stored securely in Firestore
4. Productivity score and adaptive study plan are generated
